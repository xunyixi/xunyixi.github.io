## 关于 Hexo

Hexo 是一个快速、简洁且高效的静态博客框架。它基于 Node.js 构建，可以帮助你快速创建美观的博客网站。

### 主要特点

- **快速生成**：Hexo 使用 Markdown 编写文章，一键生成静态 HTML 页面
- **主题丰富**：拥有众多精美的主题可供选择
- **插件扩展**：支持丰富的插件系统，可扩展各种功能
- **部署方便**：支持一键部署到 GitHub Pages、Vercel 等平台
